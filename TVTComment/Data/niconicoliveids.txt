# ニコニコ生放送の生放送ID設定
# ここで設定されたチャンネルはchannels.txtにも登録されている必要がある

#target	value	liveid
#

# (targetがflagsならvalueとのビットアンドが0でないなら)
# targetがnsidのときvalueは nid<<16 | sid の形式
# liveidは空白なら対応付けを解除する（！！！空白の場合でもその前のタブ文字は必須です！！！）
# 下に書いたものは上より優先される
